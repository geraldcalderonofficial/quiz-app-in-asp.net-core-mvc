# Quiz Application - ASP.NET Core MVC 
  Build ASP.NET Core MVC Web Application. Learn to create an ASP.NET MVC application from scratch and create a SQL Server database that connects to your ASPNET MVC web application via Entity Framework Core.
  
  Create A New ASP.NET MVC Project, Domain Models / Entities for EF Core, Install Entity framework core Nuget packages, Create DbContext, Add Connection string, Register DbContext to Services (Dependency Injection), Run Entity Framework Core Migrations, Seed quiz data, Create Quiz Controller, View and Submit Quiz Answers.
